# QDAP Benchmarks

> Benchmark scripts will be added in Phase 2.
> Baseline TCP metrics (iperf3) will be collected here.

## Planned Benchmarks

1. **Throughput:** 1MB, 10MB, 100MB — TCP vs QDAP-TCP vs QDAP-QUIC
2. **Latency:** 10K small messages — p50, p95, p99
3. **Multiplexing:** 10 concurrent streams — HOL blocking analysis
4. **Loss Recovery:** 1%, 5%, 10% loss — Ghost Session accuracy
5. **Mixed Traffic:** Video + audio + control — QFT strategy transitions
